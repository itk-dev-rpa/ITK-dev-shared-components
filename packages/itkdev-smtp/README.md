# itkdev-smtp

Part of the [itk-dev-shared-components](https://github.com/itk-dev-rpa/itk-dev-shared-components) package family.

SMTP email-sending helpers. Standard library only, no external dependencies.

## Installation

```
pip install itkdev-smtp
```

## Usage

This package installs into the shared `itkdev` namespace:

```python
from itkdev.smtp import smtp_util
```
