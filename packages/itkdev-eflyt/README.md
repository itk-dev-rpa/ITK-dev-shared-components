# itkdev-eflyt

Part of the [itk-dev-shared-components](https://github.com/itk-dev-rpa/itk-dev-shared-components) package family.

Selenium-based automation helpers for the Eflyt case system.

## Installation

```
pip install itkdev-eflyt
```

## Usage

This package installs into the shared `itkdev` namespace:

```python
from itkdev.eflyt import eflyt_login, eflyt_search
```
