# itkdev-misc

Part of the [itk-dev-shared-components](https://github.com/itk-dev-rpa/itk-dev-shared-components) package family.

Miscellaneous utilities: CPR/CVR validation, address lookup and file handling.

## Installation

```
pip install itkdev-misc
```

## Usage

This package installs into the shared `itkdev` namespace:

```python
from itkdev.misc import cpr_util, cvr_lookup
```
