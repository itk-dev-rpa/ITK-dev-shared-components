# itkdev-graph

Part of the [itk-dev-shared-components](https://github.com/itk-dev-rpa/itk-dev-shared-components) package family.

Microsoft Graph API helpers for reading email from shared inboxes, files and sites.

## Installation

```
pip install itkdev-graph
```

## Usage

This package installs into the shared `itkdev` namespace:

```python
from itkdev.graph import authentication, mail
```
