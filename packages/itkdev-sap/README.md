# itkdev-sap

Part of the [itk-dev-shared-components](https://github.com/itk-dev-rpa/itk-dev-shared-components) package family.

SAP GUI automation helpers for RPA projects.

## Installation

```
pip install itkdev-sap
```

## Usage

This package installs into the shared `itkdev` namespace:

```python
from itkdev.sap import sap_login, multi_session
```
